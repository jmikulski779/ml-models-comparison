# Machine Learning Models Comparison (R)

## Description
This project focuses on comparing machine learning models across different problem types:
- Binary classification
- Multiclass classification
- Regression

## Models
- k-Nearest Neighbors (k-NN)
- Decision Tree
- Neural Network

## What I did
- Implemented models and compared them with built-in R libraries  
- Evaluated performance using metrics such as Accuracy, AUC, MAE, MSE, and MAPE  
- Tuned model parameters to optimize performance  
- Analyzed differences between models across different tasks  

## Datasets
- Water quality (binary classification)
- Body performance (multiclass classification)
- Food delivery times (regression)

## Results (summary)
- Decision Trees achieved the best accuracy in classification tasks  
- Neural Networks performed best in regression  
- k-NN showed strong performance depending on parameter tuning  

## Technologies
- R
- caret, rpart, nnet
