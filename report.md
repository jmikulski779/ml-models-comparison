# Results and Analysis

## Binary Classification (Water Quality)
- Best model: Decision Tree (depth=6)
- Accuracy ~0.95
- Neural Network achieved highest AUC (~0.95)

## Multiclass Classification (Body Performance)
- Best model: Decision Tree
- AUC ~0.74
- Other models performed worse

## Regression (Delivery Time)
- Best model: Neural Network
- Lowest error across MAE, MSE and MAPE
- Decision Tree second best

## Key Insights
- Decision Trees are strong for classification problems  
- Neural Networks perform better for regression tasks  
- Model performance depends heavily on parameter tuning
